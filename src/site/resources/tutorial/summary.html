webmacro.org

You can download the most recent copy of WebMacro from its home: webmacro.org.

WebMacro is an opensource development project. That has three important implications for you:

  • You get all the source code
  • There is an active mailing list where you can get help
  • If you get involved, you might even become a developer!
At any rate, the place to start learning about WebMacro is definately the website itself.

webmacro list

You might consider joining and participating in the WebMacro mailing list. Either follow the links to it from the WebMacro website, or send mail to webmacro-request@webmacro.org to join.

You will frequently get quick and effective responses to your questions on the mailing list.

The JavaDoc API

WebMacro ships with a README and some other documentation, but the primary documentation continues to the the JavaDoc. WebMacro is largely a library, and it is best documented through its API.

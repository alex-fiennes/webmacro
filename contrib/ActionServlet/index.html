"Easy mapping of HTTP requests to methods of Java components"
ACTION SERVLET
version 0.53

Copyright (c) 1999-2001 Petr Toman

  • Basic information

    ActionServlet provides an easy mapping of user actions (HTTP requests) to methods of Java components together with automatic conversion of HTTP parameters to the values of Java types.

  • Requirements

    • JDK 1.1 or Java 2 SDK Standard Edition (or compatible),
    • JSDK 2.0 (you should be able to run ActionServlet with any servlet runtime compatible with Servlet specification 2.x like JServ or Tomcat, but examples are only pre-configured for use with JSDK 2.0).

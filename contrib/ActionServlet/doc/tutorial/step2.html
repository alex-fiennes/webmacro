Step 2/4: Write the Java code of  Authenticator component: 

public class Authenticator {
   public Authenticator(org.webmacro.as.ActionServlet as) {}

   public String login(String userName, String password) {
      // verify user name and password - here: only John may log in
      if ("John".equals(userName) && "18x79Z".equals(password))
         return "SuccessfulLogin.html";
      else 
         return "BadUserNameOrPassword.html";
   }
}

In ActionServlet terminology, login() is called action method - it will handle action invoked on HTML page - and is nested inside Authenticator class, which defines the ActionServlet component.

Note 1: Components are instantiated by ActionServlet and must have a public constructor with a single parameter of type org.webmacro.as.ActionServlet (optionally it can have a second parameter of type java.lang.String which will be passed the component name).

Note2: Action method must be defined as public and usually returns a value - an error code or a template name (see Specification for more info).

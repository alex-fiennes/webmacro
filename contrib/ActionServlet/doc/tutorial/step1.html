Step 1/4: Create the Login.wm HTML template: 

<HTML>
#set $Response.ContentType = "text/html"
<BODY>

   <FORM METHOD="POST" ACTION="$SERVLET">
      User name: <INPUT TYPE="Text" NAME="userName">
      Password : <INPUT TYPE="Password" NAME="password">
      <INPUT TYPE="Submit" NAME="action" VALUE="Login">
   </FORM>
   
</BODY>
</HTML>

In this template, there are defined names of two parameters: userName and password, which will take input user from user, and action specifying through it's VALUE action to be invoked (Login).

Note 1: It is mandatory to define action when using ActionServlet.
Note 2: $SERVLET variable is always evaluated to the actual servlet path.

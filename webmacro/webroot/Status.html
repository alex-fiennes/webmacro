WebMacro: Project Status

See the RELEASE NOTES for a list of the changes in the current version, and recent development.

WebMacro is now a stable, high performance servlet engine that is being used by a large number of sites.

The current focus for WebMacro is to improve scalability and reliability so that it can be rolled out on some of the largest sites on the net.

We intend WebMacro 1.0 to be the best performing, most scalable, most useful Java page generator on the planet, bar none.

WebMacro is being developed by a team of core developers, which you're welcome to join. Please read our POLICY document if you are interested in joining the team!

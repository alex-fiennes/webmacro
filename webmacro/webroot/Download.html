WebMacro is a Java servlet development framework intended to let people get back to what they're good at.

By cleanly separating programming work from page creation, WebMacro allows programmers to return to conventional programming in a standard language, while providing web page designers with a straight-forward way of embedding script output on a standard page.

The project status page has some details on what changed in the most current release.

Read the introduction to learn more about WebMacro. WebMacro is free software, though there are licensing terms which you should review.

Please join our mailing list and let us know what features are most important to you--these are early days for WebMacro, so at this point your ideas can make a significant contribution to the project, for which we'll give you both credit and gratitude!

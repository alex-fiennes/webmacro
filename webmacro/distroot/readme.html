WebMacro is a stable product and the ancestor of many templating projects on the net. This release is a major milestone. It proves to the Java community that WebMacro remains a foundation in the Java landscape for applications which depend on text templating. This release has undergone many months of use in development and production environments. We do not have any reported bugs in this release. Should you encounter problems/bugs, you should submit a bug report to http://sourceforge.net/projects/webmacro/.

WM is easy to use once you get the hang of it. If you are first time user, this release will not be hard to get working. If you've been using release 1.x+, you can pretty much replace your webmacro.jar with the one in this release but it is always adviseable to recompile any sources which depend on the API.

For complete information on WebMacro, this release, and usage questions, please visit the WebMacro Wiki at http://www.webmacro.org. This is the essential and most up-to-date reference for WebMacro. This document and the supporting files serve to help you get up and running quickly with WebMacro.

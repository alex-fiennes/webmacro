CGI Data:

  • Script name: $cgi.SCRIPT_NAME
  • Path info: $cgi.PATH_INFO
  • Request method: $cgi.REQUEST_METHOD
  • Path translated: $cgi.PATH_TRANSLATED
  • Query string: $cgi.QUERY_STRING

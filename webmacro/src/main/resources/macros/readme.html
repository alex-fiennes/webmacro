WebMacro Standard Macro Library

This directory contains standard, application-neutral WM macros.

  • HTML Generation Macros

  • Interfacing To PayPal and Verisign
Include the webmacro file, allmacros.wmm, to include all macros in this library: 
#include as macro "allmacros.wmm"

See http://www.webmacro.org/MacroDirective for the complete docs on how to create a macro.
